공통 이식 방법 — 시스템별 파라미터만 다름

/* 신청 폼 </head> 또는 <form> 직전에 삽입 */
<script
src="https://sdk.WageGuard.kr/v1.js"
data-system="hrd-net" /* 시스템 식별자 */
data-threshold="65" /* 고용24=60, EDI=80 */
data-signals="L1,L2,L3,L4,L5C" /* 시스템별 신호 선택 */
data-action="require-auth" /* block | require-auth | flag */
></script>
data-action 옵션
  • block — 즉시 차단 (점수 ≥ threshold)
  • require-auth — 추가 인증 요구 (권장)
  • flag — 통과 허용 + 검토 대기열
시스템별 권장 임계값
  • 고용24 실업급여: 60점
  • HRD-Net 훈련비: 65점
  • 고용보험 EDI: 80점
  • 워크넷 구직등록: 70점
적용 소요 시간
  • JS 삽입: 1~2일
  • 임계값 튜닝: 1~2주 (A/B test)
  • L5 행정신호 연동: 4~8주
  • 전체 안정화: 2~3개월
