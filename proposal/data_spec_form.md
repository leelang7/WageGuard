# 데이터 명세서 — 누리집 입력 양식

> 제5회 고용노동 공공데이터·AI 활용 공모전 누리집 (https://www.2026datacontest.co.kr/) 데이터 입력 양식에 그대로 복붙할 수 있도록 정리한 별도 양식.
> 출품작: **WageGuard — 장애인 노동 사각지대 점검 우선순위 AI**
> 부문: 제품·서비스 개발 부문
> 주관기관: 한국장애인고용공단·한국고용정보원
