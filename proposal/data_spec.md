## 1. 국세청 사업자등록 상태조회 API

| 항목 | 내용 |
|---|---|
| 데이터셋명 | 국세청_사업자등록정보 진위확인 및 상태조회 서비스 |
| 제공기관 | 국세청 |
| 데이터셋 ID | data.go.kr **15081808** |
| 엔드포인트 | `POST https://api.odcloud.kr/api/nts-businessman/v1/status` |
| 인증 | data.go.kr 일반 인증키 + 데이터셋 활용신청(자동승인) |
| 호출한도 | 100건/요청, 1,000,000건/일 |
| 갱신주기 | 국세청 원천 동기화 |
| 입력 | `b_no`: 10자리 사업자번호 리스트 |
| 핵심 응답 필드 | `b_stt`(계속/휴업/폐업), `b_stt_cd`, `tax_type`, `tax_type_cd`, `end_dt` |
| 활용 모듈 | M1(사업장 마스터), M2(휴·폐업 임박 신호) |
| 검증 결과 | ✅ 정상. 삼성/현대/카카오 = "계속사업자/일반과세자" 확인 |
| 한계 | 사업자번호를 입력으로 받아야 하므로 사업장명만으론 조회 불가 |


## 3. EIS 고용행정통계 OpenAPI — OPIB 피보험자

| 항목 | 내용 |
|---|---|
| 데이터셋명 | 고용행정통계 피보험자 현황 |
| 제공기관 | 한국고용정보원 / 고용노동부 |
| 엔드포인트 | `GET https://eis.work24.go.kr/opi/ipsApi.do` |
